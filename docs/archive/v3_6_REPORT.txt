HOW TO USE
================================================================================

1. DROP FOLDERS:
   - Drop files into data/knowledge_drop/raw_docs/, etc.
   - Run "digest" to process them
   - Files are auto-moved to knowledge store

2. BROWSER FALLBACK:
   - When DDG/Brave fail, BrowserSearchProvider tries:
     - DDG Lite (text-only)
     - Brave Search
     - Local knowledge (via ingestor)
   - If all fail → Feed Me request

3. GOLEARN:
   - Now checks local knowledge first via ingestor
   - Falls back to web search
   - Creates Feed Me requests when insufficient
