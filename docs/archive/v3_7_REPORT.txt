================================================================================
EXECUTIVE SUMMARY
================================================================================

v3.7 adds a Site Navigator Spine that enables Karma to intentionally navigate
structured sites (starting with Wikipedia) instead of relying only on fragile
search providers.

Key features:
1. Browser Agent - fetches pages and extracts content
2. Site Rules - Wikipedia-specific filtering rules
3. Bounded Navigation - max_depth, max_pages, stop reasons
4. Local Save - navigated content saved to knowledge for reuse
5. Navigation Commands - "navigate wikipedia <topic>"

================================================================================
HOW TO USE
================================================================================

1. NAVIGATE WIKIPEDIA:
   "navigate wikipedia kali linux"
   "golearn wikipedia python"
   "wiki python asyncio"

2. RESULT:
   - Pages visited and content extracted
   - Content saved to local knowledge
   - Provenance labeled as "wikipedia_navigation"
   - Available for future GoLearn runs

3. NAVIGATION POLICY:
   - Max depth: 2
   - Max pages: 6 (wikipedia default)
   - Same-site only
   - Skips: Help, Wikipedia:, Template:, Category:, etc.
   - Prefers: en.wikipedia.org only

- Stop reasons tracked (completed, max_pages)
