================================================================================
HOW TO USE
================================================================================

1. CREATE KNOWLEDGE FOLDER:
   knowledge_drive/
     00_inbox/
     01_kali_linux/
     02_python/
     03_debugging/
     04_ai_frameworks/
     05_coding_patterns/
     06_systems/
     07_docs_reference/
     08_seed_packs/

2. INGEST COMMAND:
   Run any of:
   - ingest /path/to/knowledge_drive
   - import /path/to/folder
   - seed /path/to/folder
   - karma ingest /path/to/folder

3. SEARCH LOCAL KNOWLEDGE:
   Ingestor can search by:
   - Topic: ingestor.search_by_topic("python")
   - Query: ingestor.search_local("decorator", topics=["python"])
