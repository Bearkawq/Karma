# Surgery

- Phase purpose: make precise changes in fragile or high-impact areas.
- Preferred active roles: `Surgeon`, `Faultfinder`.
- Preferred support roles: `Scout`, `Scope Guard`.
- Acceptable change size: very small.
- Common risks: hidden dependencies, confidence theater, accidental collateral edits.
- Escalation rules: escalate immediately when hidden dependencies widen the target zone.
- Success shape: exact correction with no unnecessary motion.
