# Episodic Memory

Notable past events across all projects.

## Storage

Events are stored by date:
```
episodic/
  2026-04/
    02.json
    01.json
```

## Event Format

```json
{
  "timestamp": "2026-04-02T18:30:00Z",
  "project": "nexus",
  "event": "feature_completed",
  "outcome": "success",
  "summary": "Implemented user authentication"
}
```
