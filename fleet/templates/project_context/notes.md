# Project Notes

Freeform notes specific to this project.

## Categories

- **Ideas** - Feature ideas, improvements
- **Decisions** - Architectural decisions made
- **Questions** - Things to investigate
- **Links** - Useful references

## Example

```yaml
ideas:
  - "Add WebSocket support for real-time updates"
  - "Migrate to async/await pattern"
  
decisions:
  - "2026-04-01: Use SQLite for local development, PostgreSQL for prod"
  - "2026-04-02: Skip authentication for now, add later"
  
questions:
  - "Should we use a framework or vanilla?"
  - "What's the best way to handle configuration?"
  
links:
  - "API docs: https://docs.example.com"
  - "Architecture: /docs/architecture.md"
```

## Usage

Add anything relevant that doesn't fit other .fleet files.
