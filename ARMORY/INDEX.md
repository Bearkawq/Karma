# Armory

The Armory is the shared shelf for reusable work aids. It holds a small set of loadouts, kits, playbooks, prompts, recovery steps, and upgrade lists that Builder, Checker, and Scout can use without rethinking the basics every time.

## How To Use It
- Start with `LOADOUTS.md`.
- Open one matching section index only if it helps the current task.
- Pull the smallest useful item, then get back to the work.

## How To Add Items
- Add only when the item solves a repeat problem.
- Use obvious names.
- Keep each entry short enough to scan in seconds.
- Prefer improving an existing item before adding a new one.

## How To Avoid Clutter
- Do not add one-off notes.
- Deprecate stale items in `ARMORY_STATE.md`.
- If two items overlap heavily, merge them.
