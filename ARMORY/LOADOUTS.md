# Loadouts

Pick one loadout by task type. Use the smallest matching set.

## Small Patch Loadout
- fits: bugfix, narrow edit, fast correction
- use: `Builder Prompt`, `Bugfix Kit`, `Rollback Steps`
- best for: Builder

## Bug Check Loadout
- fits: review, regression check, weak logic check
- use: `Checker Prompt`, `Regression Kit`, `Repeat Regression Response`
- best for: Checker

## Repo Search Loadout
- fits: file finding, dependency tracing, context gathering
- use: `Scout Prompt`, `Repo Discovery Kit`, `Handoff Recovery Playbook`
- best for: Scout

## Stuck Task Loadout
- fits: blocked work, repeated failure, low-confidence handoff
- use: `Helper Prompt`, `Recovery Kit`, `Stuck Task Playbook`, `Low Confidence Recovery`
- best for: Helper or the current owner

## Upgrade Idea Loadout
- fits: workflow improvement, tool upgrade, structural refinement
- use: `Dreamer Prompt`, `Validator Prompt`, `Framework Upgrade Ideas`
- best for: Dreamer and Validator
